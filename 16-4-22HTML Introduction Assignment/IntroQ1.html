b tag

It is a HTML element used to bold the text & gets the reader's attention. But, It is not a CSS element and can not use for styling. Instead of this tag, you can use the font-weigth property to bold the text or, you can use strong tag to show the importance of the text.

i tag

It is used to make the text or font italic. It is commonly uses for the special characters, words & sentences. It is also used to pay the attention of the reader.

u tag

It is also a HTML element used for highlighting. It is used to underline the text. It may be altered by CSS.

select tag

The select tag is a HTML element represents a control that provides a menu of options.

button tag

The button tag is a HTML element is used by a user with a mouse, keyboard, finger, voice command, or other technologies. It performs a actions, such that submitting a form opening a dialogue.
